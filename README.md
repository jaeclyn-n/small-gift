# 🎁 Small Gift

A simple birthday greeting website made with HTML, CSS, and JavaScript.

## ✨ Features

- 💌 Cute birthday letter animation
- 🌸 Decorative flower bouquet ornaments
- 🎂 Personalized birthday message
- 💖 Floating heart effects
- ✨ Typing text animation
- 🎊 Confetti celebration effect
- 📱 Responsive design for desktop and mobile

## 📸 Preview

This project displays a birthday card surrounded by flower bouquets with soft pastel colors and animated effects to create a warm and memorable greeting experience.

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Canvas Confetti
